SEI: Solução de Erros de Acesso

O Sistema Eletrônico de Informações (SEI) é um dos pilares da gestão documental. Erros de acesso são comuns e podem ser resolvidos com alguns passos claros. Esta seção detalha as soluções para os problemas de login mais frequentes.

Erro de Acesso com CPF

Se o usuário estiver enfrentando problemas de acesso ao SEI utilizando o CPF, recomende usar o LOGIN de rede. Em muitos casos, o login de rede é o identificador principal para acesso ao sistema.

Outros Erros de Login (Procedimento Padrão)

  1. Remover o Acesso SEI: Primeiramente, remova o acesso do usuário ao SEI através das ferramentas de administração de perfis.
  2. Desmarcar a Flag "Sist. Segurança": No perfil do usuário, desmarque a flag "Sist. Segurança".
  3. Conceder Novamente o Acesso SEI: Conceda o acesso ao SEI novamente, seguindo os procedimentos padrão de concessão de permissões.
  4. Remarcar a Flag "Sist. Segurança": Finalmente, remarque a flag "Sist. Segurança". Este processo de "reset" geralmente resolve problemas de permissão e acesso corrompidos.
